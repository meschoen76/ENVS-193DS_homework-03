# ENVS-193DS_homework-03

## General Information 

This repository contains my submission for homework 3 for ENVS 193DS! The "code" folder contains the .qmd file used to code and create the assignment. 

The 'data' folder contains both datasets for the homework, "kelp" and my personal observations.

The 'images' folder contains the two rendered plots for my slides presentation as well as the figure taken from my chosen research paper
